核心能力

Industrial IoT 部署

工业物联网绝非贴上更坚固标签的消费级 IoT。它需要可靠的硬件、确定性通信、网络安全加固，以及在恶劣环境下 7×24 小时不间断运行的能力。SURIOTA 的 IoT 部署服务覆盖全栈：

  • 边缘网关设计与部署：我们的旗舰产品 SRT-MGATE-1210 Modbus RTU/TCP 转 MQTT 网关可将传统串口数据转换为支持云端的 JSON 载荷。支持 DIN 导轨安装，工作温度范围 -40°C 至 +85°C，已在 30 多个站点经过生产验证。
  • 传感器集成：温度、振动、压力、流量、液位及定制传感器集成，支持 4-20mA、RS-485 及无线（LoRaWAN、WiFi、4G）接口。
  • 云监控面板开发：基于 AWS IoT Core、Azure IoT 或私有云的实时面板，支持告警管理、历史趋势与移动端访问。
  • 边缘计算：在边缘端进行本地数据预处理、异常过滤与协议转换，以降低带宽消耗与云端成本。

系统集成

真正的数字化转型不仅需要连接设备，更需要连接系统。SURIOTA 将传统上彼此孤立的 OT（运营技术）与 IT（信息技术）层进行整合：

  • SCADA 集成：将 Wonderware、Ignition、WinCC 或定制 SCADA 系统与上层 MES 和 ERP 平台对接。
  • MES 与 ERP 连接：实现车间设备与 SAP、Oracle 或定制 ERP 系统之间的双向数据流，支持实时生产计划。
  • OT/IT 融合：设计安全网络架构，在不影响安全与网络安全的前提下，将 Purdue 0-2 级的工厂车间网络与 3-4 级的企业网络桥接。
  • 协议转换：Modbus RTU/TCP、OPC UA、BACnet、MQTT、Profinet、EtherNet/IP——我们擅长处理多协议环境，让说不同"语言"的设备彼此对话。

云端监控与分析

原始数据若无上下文、可视化与可执行的洞察，其价值十分有限。SURIOTA 的云端监控方案包括：

  • SURGE 平台集成：与我们的 SURGE SaaS 平台原生集成，开箱即用的面板、告警与报表功能。
  • 定制面板开发：基于 Grafana、Power BI、Tableau 或完全定制的 React/Vue 面板，贴合您的 KPI 需求。
  • 告警与升级：多渠道告警（短信、邮件、WhatsApp、Telegram），支持可配置阈值、升级策略与值班轮替。
  • 历史数据与趋势：时序数据库（InfluxDB、TimescaleDB）支持多年数据保留、高速查询及审计分析所需的导出功能。

老旧系统现代化改造

更换老旧设备成本高昂、风险巨大，且往往并非必要。SURIOTA 的现代化改造方案在保留现有资产的同时赋予其数字化能力：

  • 改装与增强：为现有设备加装 IoT 传感器、边缘网关与通信模块，无需中断生产。
  • 最小停机迁移：分阶段实施，配合并行运行期、回滚计划及切换期间的 7×24 小时支持。
  • 数字孪生基础：为未来数字孪生项目奠定结构化数据采集与三维扫描基础。

案例研究

案例一：多站点制造 IoT（巴淡岛）

一家位于巴淡岛的电子制造商拥有 4 条产线，需要对设备正常运行时间、周期时间及缺陷率实现实时可视化。SURIOTA 部署了 24 台边缘网关，与其现有 SCADA 集成，并构建了定制的 AWS IoT 面板。成果：计划外停机减少 23%，6 个月内 OEE 提升 15%。

案例二：油气管道遥测（廖内省）

一家油气运营商需要对廖内省境内的 12 段管道进行远程监控。SURIOTA 安装了太阳能供电的遥测站，配备 4G 连接、压力传感器与泄漏检测。数据接入集中式 SCADA 并触发短信告警。成果：泄漏检测响应时间从 4 小时缩短至 8 分钟。

案例三：造船厂设备监控

一家大型巴淡岛造船厂需要对 18 台桥式起重机与焊接设备进行状态监控。SURIOTA 安装了振动与温度传感器，通过 LoRaWAN 无线回传。预测算法在轴承故障发生前即向维护团队发出预警。成果：起重机维护成本降低 40%，第一年内实现零计划外停机。

常见问题

典型的 IoT 部署周期需要多长时间？

单站点试点项目通常需要 4-8 周。多站点企业级推广根据规模与集成复杂度，大约需要 3-6 个月。我们始终先以试点验证架构，再推进规模化。

你们能与我们现有的 SCADA 系统集成吗？

可以。我们拥有与 Wonderware System Platform、Ignition by Inductive Automation、Siemens WinCC、Schneider Vijeo Citect 及定制 SCADA 系统的集成经验。可通过 OPC UA、Modbus、数据库连接器或 REST API 实现对接。

网络安全如何保障？

网络安全嵌入我们设计的每一层：加密通信（TLS 1.3）、基于证书的设备认证、网络分段（IoT DMZ）、远程访问 VPN 隧道及定期安全审计。我们遵循 IEC 62443 工业网络安全指南。

你们支持本地部署吗？

当然。虽然云平台具备可扩展性与较低的资本支出优势，但许多工业客户出于数据主权、低延迟或物理隔离网络的需求，更倾向于本地部署。我们可在客户基础设施上部署私有云方案（Kubernetes + InfluxDB + Grafana）。

什么是 SRT-MGATE-1210？

SRT-MGATE-1210 是 SURIOTA 自主研发的 Modbus RTU/TCP 转 MQTT 网关。它将传统 Modbus 数据转换为 JSON 格式，并发布至 AWS IoT Core、Azure IoT 或私有 MQTT 代理。支持 DIN 导轨安装，工业级温度范围，价格适合批量部署。

你们服务哪些地区？

虽然总部与工程团队位于巴淡岛，但我们服务的客户遍布印尼（巴淡岛、民丹岛、雅加达、泗水、棉兰、廖内省）及东南亚（新加坡、马来西亚、越南）。我们的团队可在整个区域内提供现场部署与支持。
