MODEL HARGA

Harga yang scale-up bersama Anda.

01 / PER-DEVICE

Berlangganan per device terhubung

Terbaik untuk deployment dengan banyak sensor. Biaya per sumber data yang prediktif, scale linear dengan footprint sensor Anda.

02 / PER-SITE

Flat-rate per lokasi

Terbaik untuk pabrik multi-lini, multi-mesin. Device tak terbatas per site, budgeting sederhana lintas organisasi.

03 / ENTERPRISE

Kontrak kustom

Untuk armada besar, multi-region, atau deployment white-label. Bicara dengan kami, kami sesuaikan terms dengan operating model Anda.
