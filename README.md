## Overview

## Setup

### Install Dependencies

```bash
npm install
```

> **Note:** Playwright & Puppeteer akan diinstall. Jika hanya perlu script Python, skip step ini.

### Playwright Setup (untuk screenshot)

```bash
npx playwright install
```

## Usage

### Backup Semua Data

```bash
node tools/js/backup-all.js
```

### Screenshot Audit

```bash
node tools/js/screenshot-audit.js
```

### Run Python Script

```bash
python tools/py/audit_insert.py
```
